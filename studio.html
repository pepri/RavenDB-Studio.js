About

RavenDB Studio.js admin UI uses HTML5, CSS3, JS, RavenDB HTTP API and famfamfam icons.

I did not like the performance of the Silverlight RavenDB Studio, so I created this as a slick alternative. RavenDB Studio.js is blazing fast and does not autorefresh the content without your action. One day, I would like to see this studio as the default console instead of the Silverlight one. You can say thank you via PayPal, if you like it.
